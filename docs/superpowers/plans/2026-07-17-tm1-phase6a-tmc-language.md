# TM-1 arc тАФ phase 6a: the `.tmc` language

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** the `.tmc` language compiles тАФ the spec's six Appendix A examples (`docs/superpowers/specs/2026-07-16-tm1-and-tmt-design.md` A.1тАУA.6) compile through `tmt compile`, link, and RUN with derivation-first goldens, including A.5's mandatory `unmapped-read` trap path. (Phase 6b adds the optimizer passes, the stdlib twins, and `-O0`/`-O1` opt-equivalence.)

**Architecture:** a front-end mirror of the `.pmc` compiler, file-for-file, in `crates/turing-machine`: lexer тЖТ lossless CST (`parse_cst`/`lower_cst` seam тАФ phase 7's fmt/lint/LSP must attach without rework) тЖТ compile (`analyze`: duplicate checks тЖТ flatten тЖТ **graft expansion** тЖТ **range expansion**) тЖТ TM IR (`TM_IR_VERSION = 1`, per-world state graphs, versioned JSON) тЖТ codegen emitting `.tma` TEXT only тЖТ the existing core assembler. Cross-world `call`s emit the declarative binding-call operand тАФ the 5b composition engine does ALL composition at link; graft bindings are applied by the COMPILER at splice time (the source-level analog of mono stamping). `wrmv` (reserved 0x12) goes LIVE first тАФ the spec's `-O0` canon compiles each rule's action to one fused `wrmv [w], [m]`.

**Tech stack:** Rust 2024; serde/serde_json (already deps); no new dependencies.

## Global Constraints

1. **The language is spec ┬з10 verbatim** тАФ 24 fully-reserved keywords (`alphabet machine tape state entry routine graph namespace export use graft bind as map with write move goto call then return stop halt debugger`; `deprecated` contextual); worlds discipline (states only inside `machine`/`routine`/`graph` braces; top level worldless; `goto` never crosses a world; every cross-world edge is `call`/`graft` with a binding); rule = classical triple `pattern -> [debugger] [write [v]] [move [v]] transition ;`; terminators `stop`тЖТ`stp`, `halt`тЖТ`hlt`, `return` only in routinesтЖТ`ret`; `entry` exactly-one-per-world on `state`/`graft`; ┬з10.8's absences stay absent.
2. **`wrmv` goes live** (dialect 0.2 тЖТ **"0.3"**): new arch-agnostic `OperandKind::WriteMoveVec` тАФ wire = TWO self-delimiting compact groups back-to-back (write vector then move vector, each 7-bit elements with the high bit terminating the group); `Operand::WriteMove { writes: Vec<u32>, moves: Vec<u32> }`; Tm1 lowering `[Write{dev:i, index} for each non-0x7F write] ++ [MoveLeft/MoveRight{dev:i} for each non-stay move]` тАФ all writes precede all moves (one formal step; behaviorally identical to the `wr; mov` pair); source form `wrmv [w...], [m...]` (write vocabulary: payload тЙд 0x7E or `-` keep; move: `<` `>` `.`); both vectors width == arity (signed-function check extends); `jm`/`jnm`-visible MR untouched. PM-1 unaffected (caps-gated / TM-only mnemonic).
3. **`-O0` canonical lowering (spec ┬з11.1, the definition):** conditional state тЖТ `rd; mtc T; djmp D` + tables; a state whose only rule is a single all-`*` pattern тЖТ straight-line code (no match), and unconditional CHAINS collapse to straight-line `wrmv`/`jmp`; `call тАж then S` тЖТ `call тАж; jmp S` (binding calls: `call <mangled> [<binding>]`); the function body starts with `ent`. Codegen NEVER emits `.rept`, `.frame`/`.map`/`.exits`, `call.m`, `retx`, or `trap` (graft-hole traps are synthesized as IR-level trap rows lowered to `trap #0`/`#1` instructions in the instance's code тАФ the ONE exception to "never emits trap": compiler-synthesized rows only, never from source).
4. **Match-table discipline is codegen's to satisfy**: exact rows SORTED lexicographically + pairwise disjoint first (codegen sorts the exact segment; MR numbering and dispatch entries are emitted together so behavior is order-independent тАФ spec ┬з4's reconciliation), partial wildcards in source order, catch-all last. Exact-row disjointness violations are FRONT-END static errors (post-expansion), never assembler surprises.
5. **Mangling / label split** (core constraint, fact-verified): world/routine/graph names mangle with `::` (namespaces) into `.func`/`call`/`.routine` SYMBOL positions (dots/colons legal there); intra-function state labels are BARE identifiers (jump labels reject `.`/`::`) тАФ state names pass through as-is (source idents are alnum+`_`), synthetic labels use a reserved pattern (e.g. `G{n}_{name}` for graft-instance internals; pick one, document, no collision with user names тАФ user idents can't collide if the scheme uses a character sequence users can produceтАж they CAN produce `G1_walk`; disambiguate by suffixing a per-world counter reserved via a check, or prefix with something the mangler guarantees unique per world тАФ implementer designs, states the collision-freedom argument).
6. **Alphabets** (spec ┬з10.2): elements glyph | glyph-range | number-range, inclusive both ends, no count form; index = position, blank = index 0 always; glyphs any non-empty UTF-8 string unique per alphabet (single-quoted; escapes `\'` and `\\` only); numeric ranges mint decimal-string glyphs; **> 127 symbols = a clear CompileError ("the n-byte symbol family is not yet implemented") тАФ recorded deviation** from ┬з10.2's "silently select the n-byte family" (the wire family doesn't exist in the toolchain; the error names the limit).
7. **Pattern bindings** (spec ┬з10.3): `as v` on enumerable elements (single value тЙб degenerate range); `* as v` forbidden; multiple bindings = cartesian product (warn above a product threshold тАФ pick 256, document); binding scope = its rule; quoted elements bind glyph-kind (`{c}` pass-through only), bare numbers bind numeric-kind (`{v┬▒k}` folded at expansion; out-of-alphabet fold = static error); char arithmetic absent.
8. **Graft expansion** (spec ┬з10.4, the compiler-side analog of mono stamping): graph-DEFINITION dependency graph acyclic (Tarjan; instance-level cycles legal); splice = copy states, substitute continuations (state names or terminators), apply the binding тАФ projection (signature order тЖТ host tape indices) + per-tape symbol maps: pattern cells map by READ-direction PREIMAGE (multi-preimage expands pattern alternatives; zero-preimage rules drop), write cells map by write direction (hole тЖТ the rule is replaced by a synthesized trap row `trap #1`); read holes synthesize prepended `trap #0` rows per ┬з5.3's taxonomy; identical instances (fragment + binding + continuations) dedup silently; `entry graft` = world entry; instance name = the spliced entry state; instance internals unaddressable. Binding grammar identical to call's (`name = tape with map { 'a'->'b', 'c'=>'d' }` / `name = state-or-terminator`); mapping omitted = identity REQUIRING equal glyph sets (ruled).
9. **`bind`** = named bound-call target, pure sugar (dedup keys on (routine, binding) regardless); `goto` on a bind name = error.
10. **TM IR**: `TM_IR_VERSION: u32 = 1`; per-world state graphs тАФ NOT a general CFG: worlds (name, tapes/arity, alphabets by index-cardinality), states (index-resolved match rows incl. synthesized trap rows, actions = write/move vectors + transition kind goto/call-then/return/stop/halt/trap), call-site binding records; serde-tagged pretty JSON, `to_json`/`from_json` round-trip + version pin tested.
11. **Static checks** (spec ┬з10.7 complete list) as spanned `CompileError`s with frozen kebab codes in pmc's house style (`line L:C: message [code]`); warnings (spec ┬з11.5: unused import/routine/graph/binding/graft-instance, undeclared external, shadowed rule) as `Diagnostic`s in `CompileReport`; `-Werror` escalates. Thin renderer.
12. **Versions**: `TMC_LANG_VERSION: &str = "0.1"` in the new parser.rs with the acceptance-contract comment mirrored; `tmt --version` gains `tmc language 0.1` (line order: tmt / tmc language / tma dialect тАФ mirror pmt's order); `TM1_TMA_DIALECT_VERSION` тЖТ "0.2" тЖТ "0.3" in the wrmv task. IR version embedded.
13. **Optimizer in 6a = the stub**: `optimizer/mod.rs` with `OptLevel`, `OptOptions`, `OptReport`, `pass_names() -> &[&str]` returning EMPTY, a fixpoint loop over zero passes тАФ `-O1` тЙб `-O0` byte-identically until 6b; CLI accepts `-O0/-O1/--fno-<pass>` (unknown pass names error against the empty registry), `--emit-ir[=STAGE]` with stages `lowered`/`final` (no `after:<pass>` matches yet).
14. **PM-1 byte-identity** (goldens; core changes caps-gated/additive); core arch-agnostic (WriteMoveVec fake-dialect-tested); no new deps; conventional commits, NO attribution footers; durable citations only (`docs/formats.md (тАж)` existing headings; new substance in prose until T8's doc step); derivation-first goldens; CLI tests in-process.
